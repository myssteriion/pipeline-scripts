remarque :
Dans l'application Jenkins, le crédential est déjà créé et se nomme "esii (debian-ic (et autres machines de déploiement))".
Le user de connexion aux machines distantes doit être "esii". Si ce n'est pas le cas, il faudra ajouter autant de crédentials dans jenkins qu'il y a de user différent.
